:hammer: CLI for read GitHub Contributions and Streaks

## Installation

```bash
npm install -g github-contribution-stats-cli
```

## CLI

```bash
github-contribution-stats <login> [-d days]
```

ex.

```bash
github-contribution-stats zaiste -d 30
```
