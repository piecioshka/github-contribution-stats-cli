## CLI

Installation:

```bash
npm install -g github-contribution-stats-cli
```

```bash
github-contribution-stats <login> [-d days]
```

ex.

```bash
github-contribution-stats piecioshka # default days=20
github-contribution-stats piecioshka -d 30
```
